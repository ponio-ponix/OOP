# OOPにおけるゲッターとメンバ変数の関係

## 基本理解
- **ゲッターメソッドが必要なのはなぜか？**  
  👉 「メンバ変数にメソッドがないから」  

- クラスの **メンバ変数（インスタンス変数）** は、  
  外部から直接アクセスできない（カプセル化のため）。  
- だから「gear.chainring」とか外から呼び出したいときは、  
  そのための **ゲッターメソッド** が必要になる。  

## Ruby の場合
- `attr_reader` は、ゲッターメソッドを自動生成してくれる糖衣構文。  
  ```ruby
  attr_reader :cog
  # ↓ 以下と同じ
  def cog
    @cog
  end
